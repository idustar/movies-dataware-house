# movies-dataware-house

Clone built with [Dva](https://github.com/dvajs/dva).

## Demo

[Live Demo](https://data-warehouse.azurewebsites.net)

## Run Locally

Install dependencies.

```bash
$ npm install
```

Start server.

```bash
$ npm start
```

Open app in browser.

```bash
$ open http://localhost:8000/
```
